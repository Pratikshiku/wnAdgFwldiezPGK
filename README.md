## Home Service Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0ba99c39800348a491282acaa9bcbf1b/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Employee), Employer, Administrator.


**Employee Functions:**
1. Login
2. Register
3. Homepage
4. Service Projects
5. Demand Information
6. Website Announcements
7. Feedback
8. Personal Center
9. My Favorites




**Employer Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Demand Information Management
6. Service Appointment Management
7. Application Appointment Management
8. Contract Signing Management
9. Electronic Signature Management
10. Employer Evaluation Management
11. Employee Consultation Management
12. Employer Consultation Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Employer Management
4. Employee Management
5. Material Certification Management
6. Project Type Management
7. Service Project Management
8. Demand Information Management
9. Service Appointment Management
10. Application Appointment Management
11. Contract Signing Management
12. Electronic Signature Management
13. Employer Evaluation Management
14. Feedback
15. System Management (Carousel Management, Website Announcements)
